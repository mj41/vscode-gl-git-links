# Change Log

All notable changes to the "gl-git-links" extension will be documented in this file.

## [0.3.1]

- Enable `gl:` link detection and navigation in all file types (previously restricted to Markdown).

## [0.3.0]

- Initial release.
- `gl:` URI scheme support (specification version 25.10.171) across Markdown, plain text, and code comments
- Repository-aware navigation with fragment support for single lines and ranges
- Diagnostics with quick fixes to normalize syntax and scaffold missing files
- Markdown preview rewrites plus editor link provider for seamless navigation
- Compact `Git Link` hover tooltips with detailed error messaging and refreshed quick-fix flow
